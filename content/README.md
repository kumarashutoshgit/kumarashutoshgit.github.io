# Portfolio Content

This folder contains all the markdown content for the portfolio website.

## Files:

- `about.md` - Introduction and personal information
- `education.md` - Educational background
- `experience.md` - Work experience
- `certifications.md` - Professional certifications
- `linkedin-recommendations.md` - LinkedIn recommendations

## How to Edit:

1. Open any `.md` file in a text editor
2. Edit the content using Markdown syntax
3. Save the file
4. The changes will be reflected when you run `npm run dev`

## Markdown Syntax:

- `# Heading 1`
- `## Heading 2`
- `**bold text**`
- `*italic text*`
- `[link text](url)`
- `- list item`
- `> blockquote`
